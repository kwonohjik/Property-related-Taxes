## 구현 패턴 참조

양도세 `TransferTaxCalculator.tsx`에 이미 완성된 패턴이 있음. 모든 계산기는 이 패턴을 그대로 따른다.

```
[Step N: 소재지 입력]
  AddressSearch 컴포넌트 → jibun 값 상태 저장

[Step M: 기준시가 입력 필드 옆 "조회" 버튼]
  → GET /api/address/standard-price?jibun={jibun}&propertyType={housing|land}&year={YYYY}
  → 응답 price → 해당 금액 필드 자동 채움
  → 실패 시: 오류 메시지 표시 + 수동 입력 유지 (fallback)
```

### T-11: 양도소득세 가이드 페이지 ✅

- **파일**: `app/guide/transfer-tax/page.tsx` (신규)
- **URL**: `/guide/transfer-tax`
- **콘텐츠 구성**:

  #### 1. 양도소득세란?
  - 정의: 부동산을 양도(매도)할 때 발생하는 차익에 부과되는 세금
  - 납세 시기: 양도일이 속하는 달의 말일부터 2개월 이내 예정신고

  #### 2. 계산 구조 (흐름도)
  ```
  양도가액
  - 취득가액 (필요경비 포함)
  = 양도차익
  - 장기보유특별공제 (최대 80%)
  = 양도소득금액
  - 기본공제 250만원
  = 과세표준
  × 세율 (6~45%, 중과세율 별도)
  = 산출세액
  - 세액공제·감면
  = 결정세액 + 지방소득세 10%
  ```

  #### 3. 1세대1주택 비과세 요건
  - 보유기간 2년 이상 (조정대상지역: 거주 2년 추가)
  - 양도가액 12억원 이하 (초과분은 과세)
  - 일시적 2주택 특례 조건

  #### 4. 세율 표
  | 과세표준 구간 | 세율 | 누진공제 |
  |---|---|---|
  | 1,400만원 이하 | 6% | — |
  | 5,000만원 이하 | 15% | 126만원 |
  | ... | ... | ... |

  #### 5. 중과세율 (다주택·비사업용토지)
  - 조정대상지역 2주택: 기본세율 + 20%p
  - 조정대상지역 3주택+: 기본세율 + 30%p
  - 비사업용토지: 기본세율 + 10%p

  #### 6. 장기보유특별공제
  | 보유기간 | 일반 | 1세대1주택 (보유+거주) |
  |---|---|---|
  | 3년 이상 | 6% | 24% |
  | ... | ... | ... |
  | 15년 이상 | 30% | 80% |

  #### 7. 절세 포인트 3가지
  1. 보유기간 2년 기산 시점 확인 (취득일 다음날)
  2. 조정대상지역 지정·해제 시점 확인 (비과세 = 취득일 기준, 중과 = 양도일 기준)
  3. 이월과세 주의 (배우자 증여 후 5년 내 양도)

  #### 8. 자주 묻는 질문 (FAQ)
