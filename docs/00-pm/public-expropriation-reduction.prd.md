# 공익사업용 토지 수용 감면 (조특법 §77) PRD

- **Feature**: `public-expropriation-reduction`
- **Owner**: transfer-tax-engine
- **Created**: 2026-04-20
- **Status**: Plan

## 1. 목적 (Why)

공익사업 시행자(도로공사·LH 등)에게 **협의매수 또는 수용**으로 양도한 토지에 대해 산출세액에서 감면해주는 제도(조특법 §77)를 엔진에 탑재한다. 이미 양도차익 계산·장특공제·비사업용 판정은 구현돼 있으나, 감면 산출 단계가 비어 있어 부재지주 임야 수용 등 실무 빈발 사례에서 최종 세액이 과대 계산된다.

## 2. 실무 기준 사례 (Must calculate)

- 서울 거주자(부재지주) 2002.5.24. 취득 임야(충남 공주시) 2023.02.16. 협의매도
- 수용가액 560,287,470 (현금 168,287,470 + 채권 392,000,000)
- 취득가 138,000,000, 필요경비 6,800,000
- 사업인정고시일 2017.4.23. → 취득 2002.5.24. ≤ 고시 2년 전 → 당연사업용(비사업용 배제)
- 감면: 현금 10% + 채권 15% 가중평균 → 산출세액에서 감면

## 3. 스코프

### In Scope
- 조특법 §77 ① 현금보상 10% / 채권보상 15% / 3년 만기특약 30% / 5년 만기특약 40%
- 조특법 부칙 §53 (2016.1.1. 이전 고시 + 2017.12.31.까지 양도 → 종전 감면율)
- 소득령 §168의14 ③ 3호 사업인정고시일 2년/5년 분기 정밀화 (2021.5.3./2021.5.4. 기준)
- 조특법 §133 감면 종합한도 1년 2억 / 5년 3억 (현재 계산건 기준 단건 한도만)
- `TransferReduction` 유니온 확장 + API zod 스키마
- UI: 양도 사유에 "수용·협의매수" 분기 + 보상금 구성·만기특약·사업인정고시일 입력

### Out of Scope
- 5년 누적 한도 실데이터 관리 (별도 이력 연동 Phase 2)
- 환지·잔여지 보상 별도 특례 (§77의2 이후)
- 상속·증여 자산에 대한 수용 연동

## 4. 핵심 UX

1. 양도 사유 선택: `일반양도` | **`수용·협의매수`**
2. 수용 선택 시 입력 노출:
   - 현금 보상액 / 채권 보상액
   - 채권 만기보유 특약: `없음` | `3년` | `5년`
   - 사업인정고시일 (DateInput)
3. 결과 카드: 감면율 구성(현금/채권 가중평균), 감면세액, 부칙 §53 적용 여부, 한도 초과 여부
