- **Source**: 양도·상속·증여세 이론 및 계산실무 (2023) / 제6편 제3장 사례 06

## 1. 배경 (Why)

환지(land replotting)된 토지는 **종전 권리면적 ≠ 환지 교부면적**인 경우가 빈번하고, **과도면적(증가분)**에 대해 청산금을 지불한 경우 **새로운 취득**으로 간주되어 동일 지번이라도 **2필지 이상으로 분리 계산**해야 한다. 현재 엔진은 한 건 양도 = 1필지 구조라 PDF 실무 사례를 재현할 수 없다.

## 3. 핵심 요구사항

### In Scope
1. **단일 양도 건 내 다필지(parcels) 분리 계산**
   - 각 필지: 취득일·취득원인(실가/환산)·취득면적·양도면적·필요경비·취득시·양도시 ㎡당 기준시가
   - 양도가액은 전체 1개, **양도면적 비율**로 필지에 자동 안분
2. **면적 차이 환산취득가**: 환산취득가 = 양도가 × (**취득면적 × 취득시단가** ÷ **양도면적 × 양도시단가**)
3. **환지처분확정일 익일 취득일**: 과도면적 취득일 = 환지확정일 + 1일 (소득령 §162 ① 6)
4. **개산공제 면적 기반**: 환산 방식 필지 = 취득시 기준시가(취득면적 × 단가) × 3%
5. **필지별 장특공제 독립 계산**: 필지별 취득일 기준 보유기간 → 30% 한도 각각 적용
6. **합산 로직**: 필지별 양도차익·장특공제 각각 산출 → 합계 양도소득금액 → 기본공제(1회) → 과세표준 → 산출세액

### Out of Scope
- 환지 이전 종전 토지의 **일부 양도** (PDF [문제2]의 305㎡ 시나리오) → Phase 2
- 권리면적 < 종전면적일 때 감소분 환가처분 (이 경우 별도 양도 간주 없이 권리면적만 잔존) → 기본 처리는 포함하나 복잡 시나리오는 Phase 2

## 4. 핵심 UX

양도 단계에 **"다필지 분리 계산"** 토글 신설 (비활성이면 기존 단필지 흐름 유지):

```
[●] 다필지 분리 계산 (환지·합병 등)

필지 목록  [+ 필지 추가]
┌─────────────────────────────────────────────────────┐
│ 필지1  취득 1996-02-18  원인 [환산▼]               │
│        취득면적 490㎡ / 양도면적 396.8㎡            │
│        취득시 단가 80,200원/㎡ (1995)               │
│        양도시 단가 709,500원/㎡ (2022)              │
│        필요경비 0원 (환산 → 개산공제 자동)          │
├─────────────────────────────────────────────────────┤
│ 필지2  취득 2007-04-27  원인 [실가▼]  [환지확정일]  │
│        취득면적 32.2㎡ / 양도면적 32.2㎡            │
│        취득가 34,000,000원                          │
│        필요경비 0원                                 │
└─────────────────────────────────────────────────────┘

총 양도가액 525,000,000원 (자동 안분 표시)
  → 필지1 485,594,405원 (396.8/429)
  → 필지2  39,405,595원 (32.2/429)
```

결과 카드:
- 필지별 양도차익·장특공제·양도소득금액 분리 표시
- 합계 양도소득금액 → 기본공제 → 과세표준 → 산출세액 흐름

## 5. 성공 지표

- PDF 원단위 앵커 8개(양도차익·장특·산출·지방세 등) 정확 일치
- gap-detector Match Rate ≥ 90%
- 기존 단필지 흐름 회귀 0건 (parcels 미제공 시 동작 불변)
- 전체 vitest 1301 → +α (신규 테스트 포함) 통과

## 6. 리스크 · 가정

- **취득시 공시지가 연도**: 취득일이 해당 연도 6월 이전이면 **전년도** 공시지가 사용 (PDF 1996-02-18 → 1995년 공시가 적용)
- **면적 합계 검증**: 필지 양도면적 합 = 총 양도면적 (교부면적) 일치해야 함
- **환산 방식 필지 필요경비**: 개산공제 자동 계산(취득시 기준시가 × 3%), 사용자 입력 불가
- **1세대1주택 · 중과세 · 비사업용** 판정은 **필지별** 속성으로 처리 (환지된 주거지역 대지는 단순 토지로 처리)
